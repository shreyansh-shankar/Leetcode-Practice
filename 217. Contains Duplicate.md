## Pattern

Hash Set / Hash Lookup

## Recognition Trigger

- Need to quickly determine whether an element has been seen before.
- Need duplicate detection.
- Need fast membership checking.

Common phrases:

```text
Contains duplicate
Already seen
Repeated element
Unique elements
Frequency lookup
```

---

## Key Insight

While traversing the array:

- If the current element has already been seen, a duplicate exists.
- Otherwise store it and continue.

Instead of comparing every pair:

```text
Store previously seen elements in a hash-based data structure.
```

This reduces lookup time from O(n) to O(1) average.

---

## State Representation

```cpp
seen
```

Where:

- `seen` contains all numbers encountered so far.    

---

## Algorithm

1. Create an empty hash set/map.
2. Traverse the array.
3. If the current number already exists in the hash structure:
    - Return `true`.
4. Otherwise insert it.
5. If traversal completes:
    - Return `false`.

---

## Accepted Code

```cpp
#include <unordered_map>
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        unordered_map<int, int> us;

        for (int num : nums) {
            if (us.count(num)) {
                return true;
            }
            us[num] = 1;
        }

        return false;
    }
};
```

---

## Cleaner Version

Since only existence matters, a set is more natural:

```cpp
#include <unordered_set>
class Solution {
public:
    bool containsDuplicate(vector<int>& nums) {
        unordered_set<int> seen;

        for (int num : nums) {
            if (seen.count(num))
                return true;

            seen.insert(num);
        }

        return false;
    }
};
```

---

## Why It Works

For every number:

```text
Have I seen this before?
```

If yes:

```text
Duplicate found.
```

If no:

```text
Remember it for future checks.
```

The hash structure allows this check in O(1) average time.

---

## Complexity

### Hash Set / Hash Map Solution

- Time: O(n)
- Space: O(n)

---

### Brute Force

```cpp
for each i
    for each j > i
```

- Time: O(n²)
- Space: O(1)

---

## Common Mistake

Using a frequency map when only existence is needed.

Example:

```cpp
unordered_map<int,int>
```

works, but

```cpp
unordered_set<int>
```

is simpler because counts are never used.
