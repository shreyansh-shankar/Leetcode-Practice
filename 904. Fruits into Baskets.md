## Pattern

Sliding Window (At Most K Distinct Elements)

Where:

```text
K = 2
```

---

## Recognition Trigger

Look for:
- Longest contiguous subarray
- At most K distinct values
- Limited categories/types
- Variable-size window

Common problems:

```text
Fruit Into Baskets
Longest Substring With At Most K Distinct Characters
Longest Repeating Character Replacement
Subarrays With K Distinct Integers
```

When you see:

```text
Longest valid contiguous segment
with limited distinct types
```

think:

```text
Sliding Window
```

---

## Key Insight

At any point we are allowed:

```text
Only 2 fruit types
```

When a third fruit appears:

```text
Keep only:
- New fruit
- Most recent fruit type
```

Everything before the most recent streak becomes invalid.

## Why `lastFruitCount` Works

This is the magic of the solution.

Consider:

```text
1 2 2 2 3
```

Before seeing:

```text
3
```

we have:

```text
lastFruit = 2
lastFruitCount = 3
curr = 4
```

Current window:

```text
1 2 2 2
```

---

Now:

```text
3
```

appears.

We cannot keep:

```text
1
2
3
```

because that would be:

```text
3 fruit types
```

---

What can we keep?

Only:

```text
2 2 2 3
```

because:

```text
2
```

is the most recent fruit type.

Its streak length is:

```text
lastFruitCount = 3
```

Therefore:

```cpp
curr = lastFruitCount + 1;
```

becomes:

```text
3 + 1
=
4
```

Exactly correct.

---

## Algorithm

For each fruit:

### Case 1

Fruit already belongs to current baskets.

```cpp
fruit == lastFruit
||
fruit == secondLastFruit
```

Then:

```cpp
curr++;
```

Window grows.

---

### Case 2

New third fruit appears.

Current window becomes:

```cpp
lastFruitCount + 1
```

because only:

```text
Recent streak
+
Current fruit
```

can remain valid.

---

### Update Fruit Tracking

If current fruit equals:

```cpp
lastFruit
```

increase:

```cpp
lastFruitCount
```

Otherwise:

```cpp
secondLastFruit = lastFruit;
lastFruit = fruit;
lastFruitCount = 1;
```

because a new streak starts.

---

## Accepted Code

```cpp
class Solution {
public:
    int totalFruit(vector<int>& fruits) {

        int lastFruit = -1;
        int secondLastFruit = -1;

        int lastFruitCount = 0;
        int curr = 0;
        int ans = 0;

        for (int fruit : fruits) {

            if (fruit == lastFruit ||
                fruit == secondLastFruit) {
                curr++;
            } else {
                curr = lastFruitCount + 1;
            }

            if (fruit == lastFruit) {
                lastFruitCount++;
            } else {
                lastFruitCount = 1;
                secondLastFruit = lastFruit;
                lastFruit = fruit;
            }

            ans = max(ans, curr);
        }

        return ans;
    }
};
```
