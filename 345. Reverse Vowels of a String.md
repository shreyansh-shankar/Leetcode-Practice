## Pattern

👉 Two Pointers (Opposite Ends)

---

## Recognition Trigger

Look for:

```text
reverse only certain characters
process from both ends
swap matching elements
```

Think:

```text
Two pointers moving toward each other
```

---

# Key Insight

We don't want to reverse the whole string.

We only want to reverse:

```text
a e i o u
A E I O U
```

Everything else should stay in the same position.

Example:

```text
hello
```

Vowels:

```text
e o
```

Reversed:

```text
o e
```

Result:

```text
holle
```

---

# Your Approach

Use two pointers:

```cpp
left  = 0
right = n - 1
```

---

## Case 1: Both are vowels

Example:

```text
h e l l o
  ^       ^
```

Swap:

```cpp
swap(s[left], s[right]);
left++;
right--;
```

---

## Case 2: Left is vowel, right isn't

```text
h e l l x
  ^       ^
```

Need a vowel on the right.

```cpp
right--;
```

---

## Case 3: Right is vowel, left isn't

```text
h x l l o
  ^       ^
```

Need a vowel on the left.

```cpp
left++;
```

---

## Case 4: Neither is vowel

```text
h x l l y
  ^       ^
```

Neither can participate in a swap.

Move both:

```cpp
left++;
right--;
```

# Complexity

### Time

```text
O(n)
```

Each pointer moves at most once across the string.

---

### Space

```text
O(1)
```

In-place swapping.

---

# Small Optimization

Currently you do:

```cpp
string("aeiouAEIOU").find(...)
```

every time inside the loop.

That creates a temporary string repeatedly.

A cleaner version:

```cpp
string vowels = "aeiouAEIOU";
```

before the loop.

Then:

```cpp
vowels.find(s[left]) != string::npos
```

# Personal Learning Note

## Pattern

Two Pointers (Opposite Ends)

---

## Recognition Trigger

```text
Reverse only some elements
Swap from both ends
Selective reversal
```

---

## Key Insight

Keep moving:

```cpp
left++
right--
```

until both pointers land on valid characters.

Then swap.

---

## Template

```cpp
while (left < right) {

    if (left invalid)
        left++;

    else if (right invalid)
        right--;

    else {
        swap(...);
        left++;
        right--;
    }
}
```

---

## Complexity

```text
Time: O(n)
Space: O(1)
```

---

## Common Mistake

Reversing the entire string first and then trying to restore consonants.

Unnecessary complexity.

Only vowels need to move.

---

## One-line Memory Hook

```text
Find next vowel from left, next vowel from right, swap them.
```
