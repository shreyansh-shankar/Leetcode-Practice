## Pattern

👉 Dynamic Programming (Bottom-Up)

---

## Recognition Trigger

Look for:

```text
Minimum jumps
Minimum operations
Minimum cost
Reach destination with least steps
```

Think:

```text
dp[i] = best answer starting from index i
```

---

# Your State Definition

You chose:

```cpp
dp[i]
```

meaning:

```text
Minimum jumps needed to reach the end
starting from index i
```

This is exactly the right DP state.

---

# Base Case

At the last index:

```cpp
dp[n-1] = 0;
```

because:

```text
Already at destination
No jumps needed
```

---

# Transition

From index `i` you can jump:

```cpp
1 ... nums[i]
```

steps.

So try all possible jumps:

```cpp
for (int jump = 1; jump <= nums[i]; jump++)
```

and take the best one.

Conceptually:

```text
dp[i]
=
1 + min(dp[i+j])
```

for every valid jump.

---

# Example

```cpp
nums = [2,3,1,1,4]
```

---

Start:

```text
dp = [INF, INF, INF, INF, 0]
```

---

### i = 3

Can jump to:

```text
4
```

So:

```text
dp[3] = 1 + dp[4]
      = 1
```

```text
dp = [INF, INF, INF, 1, 0]
```

---

### i = 2

Can jump to:

```text
3
```

```text
dp[2] = 1 + dp[3]
      = 2
```

```text
dp = [INF, INF, 2, 1, 0]
```

---

### i = 1

Can jump to:

```text
2,3,4
```

Take minimum:

```text
min(2,1,0) = 0
```

So:

```text
dp[1] = 1
```

```text
dp = [INF, 1, 2, 1, 0]
```

---

### i = 0

Can jump to:

```text
1,2
```

Take minimum:

```text
min(1,2) = 1
```

So:

```text
dp[0] = 2
```

Answer:

```text
2
```

---

# Why Your Solution Works

When computing:

```cpp
dp[i]
```

all states:

```cpp
dp[i+1]
dp[i+2]
...
```

have already been computed because you're iterating:

```cpp
from right to left
```

This satisfies the DP dependency.

---

# Complexity

### Time

Outer loop:

```text
O(n)
```

Inner loop:

```text
O(nums[i])
```

Worst case:

```text
O(n²)
```

Example:

```cpp
[1000,1000,1000,...]
```

---

### Space

```text
O(n)
```

for DP array.

# Personal Learning Note

## Pattern

Dynamic Programming (Minimum Steps)

## Recognition Trigger

```text
Minimum jumps
Minimum moves
Minimum operations
```

---

## State

```cpp
dp[i]
```

=

```text
minimum jumps needed
from index i to destination
```

---

## Base Case

```cpp
dp[n-1] = 0;
```

---

## Transition

```cpp
dp[i] =
1 + min(dp[i+j])
```

for all valid jumps.

---

## Traversal Direction

```text
Right → Left
```

because:

```text
Current state depends on future states.
```

---

## Complexity

```text
Time: O(n²)
Space: O(n)
```

---

## Common Mistake

Using:

```cpp
dp[i] = max(...)
```

instead of:

```cpp
dp[i] = min(...)
```

because the question asks for:

```text
minimum jumps
```

not maximum reach.

---

## One-line Memory Hook

```text
At each index, try every possible jump and choose the future position that requires the fewest remaining jumps.
```
