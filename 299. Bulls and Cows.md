## Pattern

👉 Hash Map / Frequency Counting

---

## Recognition Trigger

Look for:

```text
Count matching characters/digits
Handle duplicates correctly
Compare frequencies between two collections
```

Think:

```text
Frequency Map
```

---

# Problem Understanding

A **Bull** means:

```text
Same digit
Same position
```

A **Cow** means:

```text
Same digit
Different position
```

---

## Key Insight

Trying to count cows directly causes duplicate-counting problems.

Instead:

### Step 1

Count all bulls first.

```cpp
if (secret[i] == guess[i])
    bulls++;
```

---

### Step 2

For non-bull positions only:

Store frequencies.

```cpp
secretMap[secret[i]]++;
guessMap[guess[i]]++;
```

---

### Step 3

Count cows using:

```cpp
cows += min(secretMap[digit], guessMap[digit]);
```

Because:

```text
Only the overlapping occurrences can form cows.
```

---

# Example

```text
secret = "1123"
guess  = "0111"
```

---

## Bulls

```text
Index 1:
1 == 1

bulls = 1
```

---

## Remaining Digits

```text
secret:
1 -> 1
2 -> 1
3 -> 1

guess:
0 -> 1
1 -> 2
```

---

## Cows

For digit `1`:

```text
secret has 1
guess has 2

min(1,2) = 1
```

So:

```text
cows = 1
```

---

## Result

```text
1A1B
```

# Your Solution

```cpp
class Solution {
public:
    string getHint(string secret, string guess) {
        int bulls = 0;
        int cows = 0;
        int n = secret.size();

        unordered_map<int, int> secretMap;
        unordered_map<int, int> guessMap;

        for (int i = 0; i < n; i++) {
            if (secret[i] == guess[i]) {
                bulls++;
            }
            else {
                secretMap[secret[i]]++;
                guessMap[guess[i]]++;
            }
        }

        for (auto &[digit, count] : guessMap) {
            cows += min(count, secretMap[digit]);
        }

        return to_string(bulls) + "A" + to_string(cows) + "B";
    }
};
```

---

# Complexity

### Time

```text
O(n)
```

* One pass to count bulls and frequencies
* One pass through frequency map

---

### Space

```text
O(n)
```

In practice:

```text
O(1)
```

because only digits `0-9` exist.

---

# Common Mistake

Counting cows before removing bulls.

Example:

```text
secret = 1122
guess  = 1222
```

If bulls aren't removed first:

```text
same digit may be counted as both
bull and cow
```

which is incorrect.

---

# Alternative Optimization

Since only digits:

```text
0-9
```

exist, we can use:

```cpp
int freq[10];
```

instead of hash maps.

This reduces space and is the optimal solution.
