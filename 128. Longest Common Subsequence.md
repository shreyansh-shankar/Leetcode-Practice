# Key Insight

Consecutive numbers become adjacent after sorting.

Example:

```text
nums = [100,4,200,1,3,2]

Sorted:
[1,2,3,4,100,200]
```

Now the problem becomes:

```text
Find the longest contiguous streak where:

current = previous + 1
```

# Traversal Logic

Maintain:

```cpp
prev
curr
maxCurr
```

---

## Case 1: Consecutive Number

```cpp
prev - 1 == current
```

Example:

```text
4 → 3
3 → 2
2 → 1
```

Extend streak:

```cpp
curr++;
```

---

## Case 2: Duplicate

```cpp
prev == current
```

Example:

```text
5 → 5
```

Ignore duplicate.

Do NOT:

```cpp
curr++
```

Do NOT:

```cpp
curr = 1
```

Simply skip.

---

## Case 3: Break in Sequence

Example:

```text
100 → 50
```

Current streak ends.

Start a new streak:

```cpp
curr = 1;
```

# Important Bug You Found

You originally wrote:

```cpp
maxCurr = (maxCurr, curr);
```

This is:

```text
comma operator
```

NOT:

```cpp
max(maxCurr, curr)
```

So it effectively became:

```cpp
maxCurr = curr;
```

which breaks the answer.

Correct:

```cpp
maxCurr = max(maxCurr, curr);
```

## Key Insight

After sorting:

```text
Consecutive numbers become neighbors.
```

