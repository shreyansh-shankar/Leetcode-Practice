## Pattern

👉 Hash Set (Visited Nodes)

---

## Recognition Trigger

Look for:

```text
Find common node
Find first shared object
Detect overlap between linked lists
```

Think:

```text
Store visited nodes in a hash set
```

---

# Key Insight

The important thing is:

```text
Intersection means same node,
NOT same value.
```

Example:

```text
A: 4 -> 1 \
            8 -> 4 -> 5
B:    5 -> 6 /
```

Node:

```text
8
```

is literally the same memory node.

---

# Your Approach

You store:

```cpp
unordered_set<ListNode*> st;
```

Notice:

```cpp
ListNode*
```

not:

```cpp
int
```

So you're storing addresses.

---

# How It Works

Suppose:

```text
A: 1 -> 2 -> 3 -> 7 -> 8

B: 4 -> 5 -> 7 -> 8
```

where:

```text
7 -> 8
```

is shared.

---

### Step 1

Insert nodes while traversing.

```text
Set:
{1}
```

Then:

```text
{1,4}
```

Then:

```text
{1,4,2}
```

Then:

```text
{1,4,2,5}
```

Then:

```text
{1,4,2,5,3}
```

---

### Step 2

Reach node:

```text
7
```

from list A.

Insert it.

---

### Step 3

Reach node:

```text
7
```

from list B.

Now:

```cpp
st.count(pt2)
```

returns true.

Return:

```cpp
pt2
```

---

# Why It Works

The first node encountered twice must be:

```text
the first shared node
```

because once two linked lists intersect:

```text
they never separate again
```

---

# Interesting Observation

You traverse:

```cpp
pt1
pt2
```

simultaneously.

Most solutions do:

```cpp
Traverse A completely
Store nodes

Traverse B
Check set
```

Your version is a bit different but still valid.

# Personal Learning Note

## Pattern

Hash Set (Node Identity Tracking)

---

## Recognition Trigger

```text
Find shared node
Detect overlap
Check if object seen before
```

---

## Key Insight

Store:

```cpp
ListNode*
```

not values.

Because:

```text
Intersection is based on node identity,
not node value.
```

---

## Core Template

```cpp
unordered_set<ListNode*> seen;

while (headA) {
    seen.insert(headA);
    headA = headA->next;
}

while (headB) {
    if (seen.count(headB))
        return headB;
    headB = headB->next;
}
```

---

## Complexity

```text
Time: O(m + n)

Space: O(m + n)
```

---

## Common Mistake

Checking:

```cpp
node->val
```

instead of:

```cpp
node
```

Two different nodes can contain:

```text
value = 8
```

but are not an intersection.

---

## One-line Memory Hook

```text
Store node addresses; the first address seen from both lists is the intersection.
```
