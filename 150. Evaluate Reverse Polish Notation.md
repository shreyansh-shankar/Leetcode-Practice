## Recognition Trigger

Look for:

```text id="rpn1"
Postfix expression
Reverse Polish Notation
Evaluate expression
Operators appear after operands
```

Think:

```text id="rpn2"
Stack
```

---

# Key Insight

In RPN:

```text id="rpn3"
Operator comes AFTER operands
```

Example:

```text id="rpn4"
2 1 + 3 *
```

means:

```text id="rpn5"
(2 + 1) * 3
```

---

# Why Stack Works

When you see:

```text id="rpn6"
number
```

you don't know yet how it will be used.

So:

```cpp id="rpn7"
push(number)
```

---

When you see:

```text id="rpn8"
operator
```

you immediately know it should be applied to the most recent operands.

So:

```cpp id="rpn9"
pop()
pop()
evaluate
push(result)
```

---

# Your Algorithm

For every token:

### Number

```cpp id="rpn10"
st.push(stoi(token));
```

---

### Operator

Take top two operands:

```cpp id="rpn11"
n1 = st.top(); st.pop();
n2 = st.top(); st.pop();
```

Apply:

```cpp id="rpn12"
n2 operator n1
```

Then:

```cpp id="rpn13"
st.push(result);
```

---

# Important Detail

Notice:

```cpp id="rpn14"
n1 = first pop
n2 = second pop
```

For subtraction/division:

Order matters.

---

Example:

```text id="rpn15"
4 2 /
```

Stack:

```text id="rpn16"
[4,2]
```

Pop:

```cpp id="rpn17"
n1 = 2
n2 = 4
```

Compute:

```cpp id="rpn18"
4 / 2 = 2
```

NOT:

```cpp id="rpn19"
2 / 4
```

This is the most common mistake in this problem.

## Key Insight

Numbers:


## Invariant

At any point:

```text id="rpn38"
Stack contains intermediate results
for expressions processed so far.
```

---

## Core Template

```cpp id="rpn39"
for token:

    if number:
        push(token)

    else:
        b = pop()
        a = pop()

        push(a op b)
```

---

## Important Detail

Order matters:

```cpp id="rpn40"
a - b
a / b
```

So:

```cpp id="rpn41"
b = first pop
a = second pop
```

---

## Complexity

```text id="rpn42"
Time: O(n)

Space: O(n)
```

---

## Common Mistake

Doing:

```cpp id="rpn43"
b - a
```

instead of:

```cpp id="rpn44"
a - b
```

after popping.

---

## One-line Memory Hook

```text id="rpn45"
Push numbers; when an operator appears, pop two values, evaluate, and push the result back.
```
